Documentation

Resources

I created most of the assets and code myself but used several resources for the project. The Typface is Roboto being hosted through Google Fonts. I used a Favicon Maker from https://realfavicongenerator.net/. I heavily modified code from w3schools to create the dropdown menu.

Challenges

I had a lot of problems with flexbox. I had not done more than exercises and single page projects with flexbox before. Because of this I used a bunch of janky workarounds to try and solve problems that I could have avoided with using standard block/float layout. I do however feel that I have learned a lot from the process.

I also ran into issues with the method of creating parallax images I chose. The result of which is some messy repetetive css.

In addition I chose to include several components to the project including the gallery, directory, and contact info on the same page, getting all of these elements to work together was a challenge.
